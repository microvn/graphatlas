# Code of Conduct

This project follows the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

The full text is available at the link above and applies to all
contributors and maintainers in all project spaces (issues, pull
requests, discussions, and any other channel associated with the
project).

## Reporting

Report unacceptable behaviour through GitHub's private channels — see
[`SECURITY.md`](SECURITY.md) for the private reporting endpoint, or
open a private issue via the repository's Security tab.

Reports are reviewed by the maintainer and handled per the Contributor
Covenant enforcement guidelines.
